Команды для заполнения бд python manage.py <команда> (объекты создаются если их нет)

fill_db - заполняет бд тестовыми данными (также создает админа, "admin|1")

fill_account - заполняет бд тестовыми данными приложения apps.account (Сотрудники)

fill_practicum - заполняет бд тестовыми данными приложения apps.practicum (Практикум)

fill_academy - заполняет бд тестовыми данными приложения apps.academy (Академия)

fill_outsourcing - заполняет бд тестовыми данными приложения apps.outsourcing (Аутсорсинг)

create_admin - создает стандартного админа (логин: "admin", пароль: "1")
