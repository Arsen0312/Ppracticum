Команды для заполнения бд (python manage.py <команда>)

fill_db - заполняет бд тестовыми данными (также создает админа, "admin|1")

fill_account - заполняет бд тестовыми данными приложения account (Сотрудники)

fill_practicum - заполняет бд тестовыми данными приложения practicum (Практикум)

fill_academy - заполняет бд тестовыми данными приложения academy (Академия)

fill_outsourcing - заполняет бд тестовыми данными приложения academy (Аутсорсинг)

create_admin - создает стандартного админа (логин: "admin", пароль: "1")
